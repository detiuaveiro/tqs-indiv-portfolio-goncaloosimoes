
5.6 
c) Locators preferidos no Playwright Codegen: utilizar funções ARIA (como button, textbox, checkbox) combinadas com nomes visíveis ou atributos estáveis (por exemplo, id, data-test).
Fragilidade: locators baseados em textos exatos ou valores dinâmicos (como números de voo) são suscetíveis a falhas caso o conteúdo da página seja alterado.
